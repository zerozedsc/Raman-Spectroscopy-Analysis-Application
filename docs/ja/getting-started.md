# はじめに（Getting Started）

ラマン分光分析アプリケーションを使い始めるための、最小限のセットアップと基本の流れをまとめます。

## システム要件（目安）

- **OS**: Windows 10/11、macOS、Linux
- **メモリ**: 4 GB（大きなデータセットでは 8 GB 推奨）
- **画面**: 1280×720 以上（1920×1080 推奨）

## インストール方法

用途に応じて以下を選びます（詳細は {doc}`installation` を参照）。

- **実行可能ファイル / インストーラー（Windows）**: Python を入れずにすぐ使いたい場合
- **Python から実行**: 開発・カスタマイズしたい場合
