# API リファレンス（日本語版）

このページは準備中です。

現状、このプロジェクトは **外部向けに安定した Python API / CLI を提供していません**。
API としての利用を前提とした記載（import 例、クラス一覧など）は誤解を招くため、整備が完了するまで日本語版では掲載しません。

内部実装を参照したい場合は、リポジトリ内の以下のディレクトリが実質的な「API」に相当します：

- `pages/`（各画面のロジック）
- `components/`（UI部品）
- `functions/`（前処理・解析・可視化などの処理）
